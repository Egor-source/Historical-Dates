# Project README

Этот проект использует **Webpack** для сборки и **Prettier** для форматирования кода.

## Скрипты

### Запуск проекта в режиме разработки

```bash
npm run dev
```

### Сборка проекта

```bash
npm run build
```

### Форматирование проекта

```bash
npm run format
```

## Редактор контента слайдера

Позволяет удобно редактировать данные слайдов прямо в браузере и демонстрировать проект без необходимости редактирования кода.

- **Управление кнопкой открытия**
    - Находится в правом нижнем углу.
    - Скрыть кнопку можно через **правый клик (ПКМ)**.
    - Восстановить кнопку — клавиша **Escape**.
    - Кнопка видна только на ПК; на мобильных устройствах редактор не отображается.
